##Setup

1. Make sure rvm & ruby 2.4.0 is installed in your system https://rvm.io/
2. Make sure you have postgres installed
3. gem install bundler
4. Start postgres: postgres -D /usr/local/var/postgres
5. bundle install
6. rails db:create
7. rails db:migrate
8. rails s
